Arduino
Info

Arduino is an open-source prototyping platform used for building electronics projects. It consists of both a physical programmable circuit board and a software, or IDE (Integrated Development Environment) that runs on your computer, where you can write and upload the computer code to the physical board.

Types of Arduino

Arduino Uno

The Uno is a huge option for your initial Arduino. It consists of 14-digital I/O pins

LilyPad Arduino

The Lily Pad Arduino board is a wearable e-textile technology expanded by Leah “ Buechley”and considerately designed by “Leah and SparkFun”

Arduino Mega

The Arduino Mega is similar to the UNO’s big brother. It includes lots of digital I/O pins (from that, 14-pins can be used as PWM o/ps), 6-analog inputs, a reset button, a power jack, a USB connection and a reset button..

Arduino Leonardo

The first development board of an Arduino is the Leonardo board. This board uses one microcontroller along with the USB.
